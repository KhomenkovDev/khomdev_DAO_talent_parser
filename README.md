# Web3 Talent DAO Profile Parser 🌐

High-performance parser for Web3 contributors and DAO recruitment. Match candidates with on-chain project requirements using AI.

## Features
- **Web3 Skill Matching**: Automatically identifies Solidity, Rust, DeFi, and Governance expertise.
- **DAO Fit Analysis**: Evaluates contributor profiles against specific DAO project needs.
- **ATS Optimization**: Generates recommended keywords to ensure profiles pass automated filters.
- **Executive Summaries**: Brief, high-impact candidate summaries for recruiters.

## Tech Stack
- **Framework**: Streamlit
- **Parsing**: PyMuPDF (fitz)
- **AI Engine**: LangChain + Google Gemini 2.5 Flash
- **Visualization**: Altair charts

## Getting Started
1. `pip install -r requirements.txt`
2. `streamlit run ResumeAnalyzer.py`
